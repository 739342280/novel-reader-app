# 小说智读 Android App

基于 Flet 的安卓小说阅读器

## 构建状态

[![Build Flet Android APK](https://github.com/YOUR_USERNAME/novel-reader-app/actions/workflows/build-android.yml/badge.svg)](https://github.com/YOUR_USERNAME/novel-reader-app/actions/workflows/build-android.yml)

## 使用方法

1. Fork 本仓库
2. 在 GitHub Actions 中运行工作流
3. 下载构建好的 APK

## 本地开发

```bash
pip install -r requirements.txt
python main.py
```

## 技术栈

- Python 3.12
- Flet 0.84.0
- Flutter 3.27.2
